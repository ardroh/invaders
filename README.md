### Built With

- [Rust Programming Language](https://www.rust-lang.org/)

<!-- USAGE EXAMPLES -->

## Usage

To play the game simply run:
`cargo run`

## Control

- `Left` - move ship left,
- `Right` - move ship right,
- `Space` / `Enter` - fire,
- `q` / `Esc` - terminate game.
